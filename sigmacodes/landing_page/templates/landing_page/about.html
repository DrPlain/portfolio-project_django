SigmaCodes data analysis and visualization services

This project is my portfolio project built as a student of Alx. SigmaCodes is all about helping our clients tell the stories behind their data using interactive visualizations. Our web services offer our users the ability to upload their data in csv or excel format and interactively generate their desired charts. It also ensures that the appropriate charts are used for visualizations hence enabling those without statistical knowledge to still use our services.

Our team of experienced researchers are available to offer professional writing services, data analysis and research consultation. Other services we offer include professional review of research papers, data cleaning services, questionnaire structuring and lots more.

Motivation for this project

My passion for data science prompted me to find a way to automate data visualization processes such that those without indepth statistical knowledge can easily and accurately see the stories behind their data using interactive charts. Also, my love for backend web development informed my choice of using the django framework for this project.

Time frame for completion of project

This project was done in a span of 4 weeks with different tasks for each week managed using the project management software Trello board
Week 1 and 2: Front end and backend development
Week 3: Project landing page and presentation
Week 4: Code clean up
